# 🚓 Crime Case Management System (C++)

A simple and structured Object-Oriented C++ project that simulates a crime case management system. This project demonstrates how real-world entities like victims, suspects, officers, and evidence can be modeled using OOP concepts.

---

## 📌 Features

- Add victims, suspects, and evidence  
- Assign an officer  
- Generate a case report (.txt file)  
- Includes timestamp in report  

---

## 🧠 Concepts Used

- Classes & Objects  
- Vectors (STL)  
- File Handling  

---

## 🎯 Scope & Impact

### 🚨 Why this project matters in crime management

- **Structured case records**  
  Ensures victims, suspects, and evidence are organized clearly.

- **Faster report generation**  
  Automates documentation, saving time during investigations.

- **Better traceability**  
  Makes it easier to review and track case details.

- **Foundation for digital systems**  
  Represents a basic model of modern crime record management.
